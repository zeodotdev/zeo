
## MCP Server Context

You are working through Claude Code with the Zeo MCP server.

## Opening Editors

If `check_status` shows editors are closed, use the `launch_editor` tool to open them. This launches editors via the project manager in non-standalone mode, enabling cross-editor features like `pcb_sync_schematic`.

## Launching Zeo

If the MCP server fails to connect, Zeo may not be running. Launch it with: `open /Applications/Zeo.app`
